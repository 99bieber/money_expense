## Requirements

* Flutter SDK : Channel stable, 3.16.9
* Dart SDK : 3.2.6

## Get Started

**Step 1:**

Go to project root and execute the following command in console to get the required dependencies:
```
flutter pub get
```

**Step 2:**

if needed just run:
```
dart run build_runner build 
```

**Step 3:**

Run project:
```
flutter run 
```
